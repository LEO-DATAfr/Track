═══════════════════════════════════════════════════════════════════════════
     📊 HOW IT WORKS (Simple Explanation)
═══════════════════════════════════════════════════════════════════════════

Every time someone saves data:

   sdkSave(collection, id, data)  ← Existing function
              ↓
    INTERCEPTED! 🎯 (Smart wrapper)
              ↓
        ┌─────┴──────┐
        ↓             ↓
   localStorage   universalSync
   (Your app)     (Our new sync)
        ✅          ✅
        │           │
        └─────┬─────┘
              ↓
          Is Online?
            YES ✅ → Sync to Supabase ☁️
            NO  → Queue & retry later ⏸️

Result: ✅ Data always synced!
