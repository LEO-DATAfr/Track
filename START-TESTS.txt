📌 OPTION 1: Tests dans le Navigateur (Recommandé)
─────────────────────────────────────────────────

1. Ouvrez dans votre navigateur:
   → http://localhost:PORT/test-registration.html

2. Cliquez le bouton "▶️ Démarrer tous les tests"

3. Regardez les résultats en temps réel:
   ✅ Test 1: IDs autorisés
   ✅ Test 2: Inscription étudiants (3 créés)
   ✅ Test 3: Inscription professeurs (2 créés)
   ✅ Test 4: Connexion Supabase
   ✅ Test 5: Vérification localStorage

4. Pour voir les données:
   → Ouvrez la console (F12) et exécutez:
   JSON.parse(localStorage.getItem('students'))

───────────────────────────────────────────────────────────────

📌 OPTION 2: Tests Serveur (Node.js)
─────────────────────────────────────

1. Installez les dépendances (une seule fois):
   npm install @supabase/supabase-js

2. Exécutez les tests:
   node test-database.js

3. Résultats sauvegardés dans:
   /tmp/pe-hub-test-results.json

───────────────────────────────────────────────────────────────

📌 OPTION 3: Menu Interactif (Plus facile)
─────────────────────────────

bash run-tests.sh

Puis choisissez une option dans le menu.

❌ Erreur: "Où est le fichier test?"
✅ Solution: Les fichiers sont dans votre répertoire /workspaces/Track/
            • test-registration.html
            • test-database.js
            • run-tests.sh

❌ Erreur: "localhost refused to connect"
✅ Solution: Assurez-vous que PE Hub est déployée et accessible

❌ Erreur: "Node.js not found"
✅ Solution: Installez Node.js:
            curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
            sudo apt install nodejs

❌ Erreur: "Module not found"
✅ Solution: npm install @supabase/supabase-js

❌ Erreur: "localStorage empty"
✅ Solution: Les tests doivent être lancés en premier pour créer
            les données
